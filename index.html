My Game Web Service

Usage

Create player.
POST /player.php?action=create
Body params.

  • player_name

Delete player.
POST /player.php?action=delete
Body params.

  • player_id

Get player.
GET /player.php?action=get&player_id={id}
Return json object.

  • player_id
  • player_name
  • gold
  • level

Update player.
POST /player.php?action=update
Body params.

  • player_id
  • player_name
  • gold
  • level
